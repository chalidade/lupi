# About

LUPI is my first project to simplify and optimize time to create an application project. You can use this to cut off all about backend, just focused on front-end then all about back-end give it to LUPI to solve them. Lupi can modified as you want, this seems like starter pack for you to develope an application. Lupi using lumen, if you're using laravel you will easy to use this
